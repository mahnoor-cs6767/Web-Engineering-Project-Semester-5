---

## 🧠 Project Description

This project demonstrates the design and development of a **multi-page educational website** named **Little Learners**.  
It was developed as part of the **Web Engineering course** to gain hands-on experience with website design, frontend development, and backend connectivity.  

The project includes:
- A **Home page** with navigation bar, hero section, and footer  
- **Cards section** (Stories, Hadith, Duas, Words)  
- **Games section** with interactive slider  
- **Quiz section** with multiple categories (Islamic, Science, GK, English, Math, Pak History)  
- **Login & Create Account pages** with database connectivity using PHP & MySQL  

---

## 🛠️ Technologies & Tools Used

- **Frontend:** HTML5, CSS3, JavaScript  
- **Libraries:** Font Awesome, Swiper.js, AOS Animation  
- **Backend:** PHP (db.php for connectivity)  
- **Database:** MySQL (`little_learner` schema)  
- **Assets:** Images, icons, and custom styling
